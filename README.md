# TJUPT UserScript

面向 TJUPT 站内用户的油猴脚本合集。

## 脚本列表

### [qBittorrent TJUPT 详情直达](./qbittorrent-tjupt/)

在 qBittorrent WebUI 中选中 TJUPT 种子时，自动在属性面板 Tab 栏末尾追加「🔎种子详情」和「⌛️H&R考核」两个快捷入口，一键跳转至对应页面。

## 安装

所有脚本均需配合 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展使用，支持 Chrome / Firefox / Edge。
